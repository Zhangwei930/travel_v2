# 出游助手 — 12 屏详细设计规范

> 每屏对应 `screens/` 下的同名参考图，开发时**以参考图为准，本文档为辅助说明**。

---

## 屏 01 — 启动页 (Splash)

**路径**：App 启动后首屏，2 秒后跳转 02 定位页

**布局**（自上而下）：
1. 系统状态栏 (9:41 居左、信号/WiFi/电池居右)
2. 留白 ~120px
3. 主标题「出游助手」— 44px / 800 / 深绿 `#0F5E4D` / 字距 2px / 居中
4. 副标题「发现身边好去处」— 18px / 400 / 灰 `#9CA3AF` / 居中
5. 等距视角地图插画 (绿色园林 + 蓝湖 + 中央定位 pin) — 约 280×260
6. 机器人吉祥物（深绿头 + 白身 + 蓝/黄眼睛 + 顶端天线）— 约 100×100
7. 底部 caption「正在为你准备附近好去处...」— 16px / 400 / 灰

**资源**：`source/assets/splash-cropped.png` 是已裁好顶部状态栏的整图，可直接当 background 用，也可拆成「标题 + 插画 + 机器人」三块独立资源以便后续替换。

## 屏 03 — 首页 (Home) ⭐ 核心

**Header (白底)**：
- 左：「出游助手」(24px / 800 / 主绿) ／ 📍「成都市 · 春熙路附近」(14px / 500)
- 右：🔍 ／ ☀️「晴 22°C」(14px) ／ 👤「3~5km 推荐范围」(13px / muted)

**入口大卡块 (浅绿底 `#F2F7F4` / 圆角 16 / margin: 12px 14px)**：
- 顶部：机器人头像 (50×50) + 「出游助手」(20/800) + 「已根据你的位置，为你准备了4种出游方式」(13/textSub)
- 4 张小卡 (2×2 网格)，白底 + 圆角 12 + 小阴影：
  1. 📍 按场所索引 / 景点·周边·趣游
  2. 🏢 附近现在适合去 / 短距离·即刻推荐
  3. 🛍️ 精选路线 / 2小时·半日·一日
  4. 💬 直接咨询 / 告诉我需求
- 每张小卡：32×32 彩色图标 + 标题(14/800) + 副标题(11/muted) + `›`

**"附近现在适合去" 列表 (白底卡块)**：
- 标题 + 「更多 ›」
- 每行：72×60 缩略图 + 标题(15/700) + 行内绿色标签(12) + 距离(13/绿/600) + ⭐评分

**"精选路线" 横滑列表**：
- 标题 + 「更多 ›」
- 3 张卡 (150×180)：
  - 「2小时轻松路线」绿色渐变底 + 含树木图
  - 「半日文艺路线」暖色渐变底 + 咖啡街景
  - 「一日畅玩路线」山水蓝绿底
- 卡内：标题(15/800) + 副(11/text/0.75) + 底部 📍距离(12/绿/600)

**底部 TabBar**：出游 (active) / 发现 / 咨询 / 我的

## 屏 06 — 附近现在适合去 (Nearby)

同 05 结构，但：

- Filter：全部 / **推荐** / 最近 / 免费 / 室内（注意「推荐」是带绿色光晕的active状态，可双激活）
- 底部 TabBar：active = 出游

## 屏 07 — 精选路线 (Routes)

- NavBar：← / 「精选路线」 / 副「多种时长路线组合」

  - 右下距离 (14/green/600)

## 屏 08 — 路线详情 (Route Detail) ⭐ 含地图

- NavBar：← / 「2小时轻松路线」
- 标签行：轻松 / 亲子 / 适合下午 / 低预算 (chip 风格)
- 描述 (14/textSub)
- 状态行：⏰约2小时 · 👥3个地点 · 📍约3.2km
- **地图块** (高度 220 / 圆角 14 / 1px 边)：
  - 河流、道路、公园色块、树点
  - 3 个红色编号 pin (1/2/3) 连绿色路径
  - 文字注记：人民公园、太古里、四川博物馆、宽窄巷子等
- 「路线地点」列表：3 行
  - 32 圆深绿编号 + 名称(16/700) + 描述(13/muted) + 距离
- 底部固钉 CTA 按钮：「跟着路线导航」(56 高 / 圆角 30 / 深绿底 / 19/800 / 字距 2)

## 屏 09 — 地点详情 (Place Detail)

- NavBar：← / 「地点详情」 / ↗ 分享
- Hero 大图 (h 210 / 圆角 16 / margin 14)
- 主块：
  - 标题「人民公园」(30/800)
  - 右侧：⭐ 4.7 分
  - 信息条「公园 ｜ 免费 ｜ 1.2km」(15/绿)
  - 简介 (15/text/lh 1.7)
  - 标签 chip 行（圆角 16 / 浅绿底）
- 信息表 (每行 16/0 padding / 底分割线)：
  - 地址 / 开放时间 / 适合人群 / 建议时长（左标题用绿色）
- **底部固钉操作栏**：
  - ⭐ 收藏 / ↗ 分享 / 「导航去这里」(主绿按钮，✈ icon)

## 屏 10 — 咨询·开场 (Chat Empty)

- NavBar：← / 「出游助手」 / 🔍
- 「📍成都市 · 春熙路附近」(14/绿/500)
- 机器人头像 (44) + 灰色气泡「你好呀！我是出游助手，有什么可以帮你的吗？」
- 5 个建议气泡 (浅绿底，圆角 22)：
  - 带孩子2小时内去哪玩?
  - 雨天室内去哪?
  - 夜晚适合去哪里?
  - 低预算去哪?
  - 附近适合约会吗?
- 底部输入栏：圆角 25 + 占位「问问附近适合去哪...」+ 😊 表情 + ⊕

## 屏 11 — 咨询·推荐 (Chat Reply)

同 10 框架，但：
- 用户消息靠右（浅绿气泡 + 灰头像）
- 机器人回复：先气泡描述「根据你的位置，推荐这几个适合亲子游玩的地方：」
- 然后 1 个浅灰大气泡，内嵌 3 张白色推荐卡：62×56 缩略图 + 名称(15/700) + 描述(12/muted) + 深绿「导航」按钮

---

## 屏 12 — 我的 (Profile)

**用户头部 (浅绿底圆角 16)**：
- 70 圆白头像 (含深绿 user icon)
- 「游客用户」(20/800) + 「点击登录/注册」(14/绿/500)
- 右侧：齿轮按钮 (40 圆 / 浅灰底) + `›`

**统计卡 (白底圆角 16)**：3 列分割
- 12 / 收藏
- 23 / 足迹
- 5 / 路线
- 数字 24/800、标签 13/muted、列间 1px 灰线

**常用功能 (白底圆角 16)**：
- 标题「常用功能」(17/800)
- 2×4 网格，每格：54 圆 (浅绿底) + 绿色 stroke SVG icon + 标签(12/text)
- 6 个功能：我的收藏 / 浏览历史 / 我的路线 / 离线地图 / 意见反馈 / 设置

底部 TabBar：active = 我的
